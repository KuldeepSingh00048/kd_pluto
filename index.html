Pluto Personal Assistant

Pluto assistant is a personal assistant like cortana that recognise our voice and perform the different types of task which s implented to it. This project implented only in the python programming language. Pluto will accept the voice of the user and solve their problems by showing them and speaking them the required results that users want.
The objective of this project is to enable computers to perform such intellectual tasks as decision making , problem solving, perception understanding human communication.
To replace human efforts from
1. Repetitive tasks
2. Human life critical life like NASA mission
3. Making health care doman more effcient.
